Links

Podemos introducir hipervinculos en nuestra pagina a traves de la etiquete 'a'. Detras de esta viene el atributo href, donde le indicaremos a que pagina web queremos acceder

Por defecto, un enlace aparecera si no ha sido visitado aparecera subrayado y en azul, mientras que si ha sido visitado aparecera en morado. Por ultimo si esta es uso aparecera en rojo.

Podemos cambiar los colores de este mediante css:

  • a:link + style color: Cambiamos el color del link no visitado.
  • a:visited + style color: Cambiamos el color del link visitado
  • a:active + style color: cambiamos el color de un link activo

Atributo de destino

Podemos indicar en donde queremos que se abra el enlace de nuestra que hemos insertado usando el tag target, podemos habirlos de diferentes maneras:

  • _Blank: Abre el documento en una nueva ventana.
  • _self: Abre el documento en la misma ventana en la que si hizo clic, esta opcion es predeterminada.
  • _parent: abre el documento en el marco primario.
  • _top: abre el documento vinculado en el cuerpo completo de la ventana.

Mapas de imagen

Un mapa de imagen es una imagen con areas en las que se puedee hacer clic y ver informacion. Para poder utilizar esto debemos de añadir el atributo usemap a la etiqueta de la imagen.
Para crear el mapa se utiliza la etiqueta map, y le damos un nombre con el atributo map, que debera ser el mismo que usemap de la etiqueta img. Una vez creado el mapa, debemos de delimitar las areas, que pueden ser de diferentes maneras:

  • rect: define una region rectangular
  • circle: define una region circular
  • poly: define una region poligonal
  • default: define toda la region
Una vez establecico que tipo de area queremos, le introducimos las coordenadas de la parte de la imagen.

Imagenes de fondo

Para agregar una imagen de fondo, debemos de usar un estilo en particular, el background-image, seguido del atributo url, donde le indicamos la ruta de nuestra imagen. Si queremos que toda la pagina contenga la imagen de fondo, colocaremos el estilo background-image indicando al elemento body. Si por un casual nuestra imagen es mas pequeña que el elemento en cuestion, se repetira una y otra vez. Si no queremos que la imagen se repita, debemos de usar la propiedad background-repeat, indicandole con no-repeat. Si queremos que la imagen de fondo cubra todo el elemento, se puede establecer la propiedad background-size en cover. Además, para asegurarnos de que todo el elemento esté siempre cubierto, se utiliza la propiedad background-attach en fixed.
Si por el contrario queremos que la imagen se estire, en la propiedad background-size e indicarle las medidas.

Font-awesome

Font awesome es un framework de iconos vectoriales y estilos css. Se utiliza para sustituir imagenes de iconos comunes por graficos vectoriales convertidos en fuentes. Para ellos utiliza una libreria de mas de 400 iconos. Esto nos ayuda a reducir el numero de imagenes de nuestra pagina, lo que conlleva en una mejor velocidad de nuestra web. Ademas de esta ventaja, tambien nos ofrece ventajas, como que no es necesario el uso de JavaScript para insertar estos iconos. Tambien nos permite escalar nuestros conos sin problemas de resolucion. Es totalmente gratuito.

Tablas

Una table se define con la etiqueta table. En ella podemos tener un numero de filas que se introducen a traves de la etiqueta tr. Si queremos que esa fila sea el encabezao de debemos usar th, que por defecto estan en negrita y centrados. Para crear cada una de las celdas debemos de utilizar la etiqueta td, que son los contenedores de datos de la tabla, pueden contener cualquier elemento visto anteriormente.
Por defecto, las tablas apareceran sin bordes, si queremos establecer borders, debemos de añadir el estilo border, indicando el numero de pixeles, como sera, y el color. Si queremos que los bordes sean contraidos, debemos de introducir el estilo border-collapse:collapse.
Otro estilo que podemos de introducir en nuestra tabla es el relleno de las celdas, es decil el espacio entre el contenido y sus bordes. Para ello utilizamos la propiedad padding. Ademas tambien podemos ed agregar un espacio entre las celdas, para ello utilizamos la propiedad border-spacing.
Algo muy comun es que una celda ocupe mas de una columna, es decir, unir dos o mas celdas. Si queremos hacer esto debemos de usar el atributo colspan, e indicarle cuantas columnas queremos que abarque. Por el contrario, si queremos que una celda abarque mas de una fila, usamos el atributo rowspan, indicando el numero de filas que queremos que abarque.
