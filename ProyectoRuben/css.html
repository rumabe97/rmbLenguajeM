CSS

En esta pagina vamos a poder visualizar toda la informacion sobre CSS que veremos a lo largo del curso. Ademas lo iremos ampliando poco a poco.

Como agregar elementos a html

  1. En linea: Se usa mediante el atributo Style en los elementos html, es poco recomendable. Se utiliza en un unico elemento Html
  2. Interno: Se usa mediante el uso de la etiqueta Style en la seccion del Head. Se utiliza en usa sola pagina de Html
  3. Externo: Se crea un archivo externo con el extension css. Es la manera mas recomendable de hacerlo. Se puede utilizar para definir muchas paginas de Html. Se introduce mediante le etiqueta link seguida de rel="stylesheet" href="nombre.css"

Estilos CSS

En el Css se deben de introducir todos los estilos que queremos que tenga nuestra pagina web. Algunos ejemplos son:

		Color: Define el color que tendra un determinado texto.
		Font-family: En este apartado definiremos que tipo de letra queremos tener en nuestra pagina.
		Font-size: En este apartado colocaremos el tamaño del texto de nuestra pagina.
		Bordes: Son lineas que hacen un marco al rededor del apartado donde lo coloquemos, para ello utilizamos la siguente sintaxis: Border: 2px solid red.
		Relleno: Esto se utiliza con el paddinf, sirve para agrandar pixeles hacia todos los lados de un elemento, utilizamos la etiqueta padding:50px;
		Margen: Lo utilizamos para colocar margenes en un elemento, lo utilizamos con la etiqueta margin:60px;

Float

La propiedad Float te perminte posicionar y formatear contenido, es decir, colocar una imagen o bloque que flote a la izquierda de un elemento o derecha.
Float puede tener uno de estos valores: 

		Left: El elemento flota a la izquierda de su contenedor.
		Right: el elemento flota a la derecha de su contenedor.
		None: El elemento no flota, viene por defecto.
		Inherit:  Hereda el valor de su padre.
